Bonsai 1-bit LLM (1.7B, Qwen3 arch) compiled from Almide to WebAssembly. Pure Almide implementation — every primitive (GGUF parser, Q1_0 decoder, RoPE, GQA attention) comes from the language, not a JS ML framework.

L3 numerical correctness has been verified on the native Rust target: prompt "The capital of Japan is" (tokens [785, 6722, 315, 6323, 374]) produces next-token id 26194 (" Tokyo") — byte-for-byte identical to llama.cpp on the same GGUF.

WASM path currently decodes individual tensors correctly; the 28-layer forward pass is blocked on two more codegen items listed in docs/WASM_ROADMAP.md (live Q1_0 matmul kernel + a memory-frugal matrix.to_lists).
